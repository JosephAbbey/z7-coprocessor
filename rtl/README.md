# RTL files

## AXI MM Interface for operations

### Binary operations

- Registers:

  1. Operand A (32 bits)
  2. Operand B (32 bits)
  3. Result (32 bits)
  4. Unused (32 bits)

### Unary operations

- Registers:

  1. Operand (32 bits)
  2. Unused (32 bits)
  3. Result (32 bits)
  4. Unused (32 bits)

### Result only

- Registers:

  1. Unused (32 bits)
  2. Unused (32 bits)
  3. Result (32 bits)
  4. Unused (32 bits)
